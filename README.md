# HugMe App 💖

HugMe is a prototype mobile application focused on mental health support. This project was developed as part of a Mobile Programming assignment.

## 📱 Features

* Welcome Page
* Login Page
* Sign Up Page
* User Data Form
* Home Page (mental health UI concept)

## 🧠 App Concept

HugMe is designed to help users monitor and improve their mental well-being through features such as:

* Mood tracking
* Simple journaling
* Panic support (breathing & grounding techniques)
* Activity recommendations
* Mental health statistics

## 🛠️ Tech Stack

* Kotlin
* Android Studio
* XML Layout

## 📌 Project Status

This project is currently in the early development stage (UI/Frontend only).
There is no database or backend integration yet.

This application is created as a prototype to fulfill the requirements of a Mobile Programming assignment.
